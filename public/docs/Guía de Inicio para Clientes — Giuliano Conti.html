Guía de Inicio para Clientes

Para que tu sitio web sea rápido, seguro y de tu propiedad, necesitamos configurar las cuentas correctamente. Esta guía aplica al modelo de Pago Único.

Aplica a: Pago Único
¿Contrataste el plan Mensual? No necesitás hacer nada de esto. Yo me encargo de toda la infraestructura — hosting, base de datos, dominio y mantenimiento. Vos solo comprás tu dominio y me lo informás.
1
github.com

GitHub — Donde vive el código

GitHub almacena el código fuente de tu sitio de forma segura. Es la base de todo el sistema.

  • → Entrá a github.com y hacé clic en "Sign up".
  • → Creá la cuenta con tu correo corporativo o personal.
  • → Confirmá tu correo electrónico (revisá la bandeja de entrada).
  • → Guardá bien el usuario y contraseña en un lugar seguro.
2
vercel.com

Vercel — Donde se ve tu web

Vercel publica tu sitio en internet y lo hace ultrarrápido. Se conecta directamente a GitHub.

  • → Entrá a vercel.com.
  • → Seleccioná "Continue with GitHub" (no crees cuenta con email).
  • → Autorizá la conexión cuando te lo pida.
  • → Si te pide número de teléfono para verificación, ingresá el tuyo (es solo para seguridad).
Importante: Usá siempre "Continue with GitHub" para que ambas cuentas queden vinculadas automáticamente.
3
supabase.com — Solo para Business

Supabase — Tu base de datos

Supabase guarda de forma segura los datos de tu aplicación (usuarios, formularios, productos, etc.). Solo aplica si tu proyecto incluye base de datos.

  • → Entrá a supabase.com.
  • → Hacé clic en "Sign in" y seleccioná "Continue with GitHub".
  • → Aceptá los permisos solicitados.
4
analytics.google.com — Opcional

Google Analytics — Tus estadísticas

Te permite ver cuántas personas visitan tu sitio, desde dónde y qué páginas miran.

  • → Iniciá sesión en analytics.google.com con tu Gmail.
  • → Hacé clic en "Empezar a medir".
  • → Ponele el nombre de tu empresa y seguí los pasos.
  • → Compartí acceso de Administrador con: tech@giulianoconti.com
Nota legal: Es importante que Analytics esté a tu nombre. Vos sos el dueño de los datos de tus visitantes — yo solo tengo acceso para la implementación técnica.

¿Qué sigue después de crear las cuentas?

Acceso completo

Enviame por un canal seguro (WhatsApp, email) el usuario y contraseña de GitHub. Yo ingreso, configuro Vercel y Supabase, y al terminar te aviso para que cambies todas las contraseñas.

Sin compartir contraseñas (+20%)

No necesito tus claves. Solo decime tu nombre de usuario de GitHub y te envío una invitación de colaborador. Haremos la configuración de Vercel juntos por videollamada.

Sobre los costos de las plataformas: En todos los casos se usan los planes gratuitos de GitHub, Vercel y Supabase — son muy potentes y suficientes para la mayoría de los proyectos. Si en el futuro tu web crece mucho, estas plataformas te avisarán antes de cobrar algo.
